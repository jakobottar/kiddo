# Vision Mamba

[Vision Mamba](https://arxiv.org/abs/2401.09417) test and implementation repo

## Installation

install `conda` env with 

```bash
conda env create -f env.yaml
```

then install `mamba-ssm` from local copy with 

```bash
pip install -e mamba-1p1p1
```
